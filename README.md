# TCP Lock-Free Server/Client

Проект на C++20 реализует простой TCP сервер и клиент с многопоточностью. Основная цель — продемонстрировать работу с сокетами, потоками и lock-free подходами к обработке данных.

---

## Структура проекта

- `server/` — исходники сервера  
  - `server.cpp` / `server.h` — основной класс сервера  
  - `main.cpp` — точка входа сервера  
- `client/` — исходники клиента  
  - `client.cpp` / `client.h` — основной класс клиента  
- `common/` — общие заголовки и утилиты  
- `CMakeLists.txt` — CMake скрипт сборки проекта  

---

## Зависимости

- C++20 компилятор (GCC >= 13, Clang >= 15, MSVC >= 2022)  
- CMake >= 3.10  
- Linux или WSL/Windows для сборки
